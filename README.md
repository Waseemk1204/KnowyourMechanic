# KnowyourMechanic 🚗🔧

A modern web platform that connects customers with verified garages in Pune. Find reliable mechanics, get instant quotes, book repairs, and manage your vehicle services all in one place.

## Features

- 🔍 **Find Verified Garages** - Browse verified garages on an interactive map or list view
- 📋 **Smart Questionnaire** - Fill out a simple questionnaire to get instant cost estimates
- 💬 **Video Consultations** - Get expert advice through video calls before visiting the garage
- 💳 **Secure Payments** - Integrated payment gateway for safe and secure transactions
- ⭐ **Verified Reviews** - Read reviews from verified customers who completed service
- 📱 **Mobile Responsive** - Works seamlessly on desktop, tablet, and mobile devices

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Maps**: Leaflet
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Waseemk1204/KnowyourMechanic.git
cd KnowyourMechanic
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint
